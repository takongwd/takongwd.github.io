# Takong Wedding Laos 📸

Fine Art Wedding Photography & Photo Booth Services website.

**Live Site:** [https://takongwd.github.io](https://takongwd.github.io)

## Tech Stack
- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Database & Backend:** Supabase (PostgreSQL)
- **Notifications:** Telegram Bot API
- **Client Communication:** WhatsApp Integration

## Features
- Real-time Availability & Booking Calendar
- Bot protection via Honeypot trap & Math CAPTCHA verification
- 2026 Price Packages & Photo Booth Add-ons
- Metallic Gold Luxury Branding theme
- Smooth logo reset & scroll-to-top interaction
- Right-click image download prevention overlay shield
